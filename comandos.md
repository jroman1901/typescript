Instala Node.js y npm:

Descarga e instala Node.js

node -v
npm -v

Instala TypeScript:

Una vez que tengas Node.js y npm instalados, puedes instalar TypeScript globalmente en tu sistema utilizando el siguiente comando:

npm install -g typescript
